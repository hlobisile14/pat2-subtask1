# Morse Code

## 1. Overview of Morse Code
Morse code is a method of encoding text characters as sequences of two signals: dots (.) and dashes (-). It was developed to transmit information over long distances using telegraph lines. Each letter, number, and punctuation mark has a unique sequence of dots and dashes.

## 2. Historical Context
Morse code was invented in the 1830s by Samuel Morse and Alfred Vail. It was created to be used with the electric telegraph, which was the first form of long-distance electronic communication. The system became widely used in the 19th and 20th centuries for railways, maritime communication, and military operations because it could transmit messages quickly and reliably over wires and later radio.

## 3. How the Morse Code System Works
The system uses only 2 elements:
- **Dot (.)**: A short signal
- **Dash (-)**: A signal that is 3 times longer than a dot
- **Space between letters**: Equivalent to 3 dots
- **Space between words**: Equivalent to 7 dots

Example of timing: SOS = `... --- ...`

### Example Translations
| Text | Morse Code |
| --- | --- |
| HELLO | `.... . .-.. .-.. ---` |
| WORLD | `.-- --- .-. .-.. -..` |
| SOS | `... --- ...` |
| NCV PAT | `-. -.-. ...- .--. .- -` |
